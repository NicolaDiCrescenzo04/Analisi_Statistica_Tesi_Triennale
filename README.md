# Analisi_Statistica_Tesi_Triennale
Analisi statistica dell'impatto della struttura del capitale sulla redditività delle aziende ICT (Software &amp; Hardware) negli Stati Uniti. Il progetto replica ed estende la metodologia di Kim et al. (2023) applicandola al mercato americano attraverso un'analisi multi-anno e modelli di regressione lineare multipla.
